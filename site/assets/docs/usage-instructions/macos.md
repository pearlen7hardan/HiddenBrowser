# macOS

## Programs for proxy setup on macOS

Clients for routing traffic through a SOCKS5 / HTTP proxy. Pick one, import your proxy, and point the app at it.

### [<mark style="color:purple;">**V2Box**</mark>](https://apps.apple.com/us/app/v2box-v2ray-client/id6446814690)

A V2Ray / Xray proxy client for iOS, macOS and Android. Imports a SOCKS5 proxy from a link or config file and tunnels traffic, including UDP. Free — distributed on the App Store.

* Platform: iOS, Android, macOS
* UDP support: Yes
* Difficulty: Medium
* Split routing: Added in recent updates
